Privacy Policy

Effective date: 2025-11-20

1. Introduction

"What's the Point — AI Summarizer" ("the extension", "we", "us") provides on-demand article and webpage summaries using an AI service. This Privacy Policy explains what data we collect, why we collect it, how we use it, and what choices you have.

2. Data we collect and why

  • Page content (article text, title, URL) — Collected only when you explicitly request a summary by using "Summarize Current Tab". This content is sent to OpenAI to generate the summary.
  • Local data — We store a daily usage counter (number of free summaries used today) and a license-valid flag (after verifying a Gumroad license) in chrome.storage.local. We do not store summaries or page contents locally in this version.
  • License key (optional) — If you paste a Gumroad license key to verify your subscription, it is sent to Gumroad's verification API. We do not publish your license key; it is only used for verification and (optionally) stored locally as a flag.

3. Purpose of processing

  • To generate the summary you requested (we send the page content to OpenAI).
  • To verify license status with Gumroad.
  • To manage your local usage quota (free-per-day counter).

4. Third parties

When you request a summary, the requested page content is sent to OpenAI for processing. If you provide a license key, it is verified via Gumroad. Please consult OpenAI and Gumroad privacy policies for details on their data handling.

5. Retention

This extension does not retain page content locally. Any retention of data by OpenAI or Gumroad is controlled by those services' policies. Local storage (counters and license flags) remains until you clear extension data or uninstall the extension.

6. Security

All outgoing requests to external services use HTTPS. Important: this build embeds the OpenAI API key into the extension package. Embedding keys client-side is inherently insecure because someone with the extension package can read it. We recommend a server-side proxy for production to keep API keys secret.

7. Permissions

  • activeTab — Temporary access to the active tab when you request "Summarize Current Tab".
  • storage — For storing local counters and license flags.

8. Your choices

You may uninstall the extension at any time to remove local data. To avoid sending page content to OpenAI, do not use the Summarize feature. You can clear extension storage in Chrome's site settings.
